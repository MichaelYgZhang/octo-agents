# 复盘总结报告功能

## 功能概述

复盘总结报告模块用于分析每日预测准确率，识别预测差异原因，并生成改进建议反馈给Agent系统。

## 主要特性

### 1. 多周期复盘
- **每日**：分析当日预测准确率
- **每周**：周度总结预测表现
- **双周**：两周滚动复盘
- **每月**：月度综合分析
- **季度**：季度大总结
- **年度**：年度复盘报告

### 2. 预测差异分析
- 自动识别误差超过3%的预测
- 分析差异原因（市场情绪、突发事件、系统性风险等）
- 提供具体改进建议

### 3. Agent学习反馈
为每个Agent生成个性化反馈：
- **量化分析师**：技术指标优化建议
- **基本面分析师**：估值模型改进建议
- **新闻分析师**：情感分析优化建议
- **风险分析师**：风险预警改进建议

### 4. 改进建议沉淀
- 自动生成投资策略改进建议
- 将经验教训沉淀为文档
- 持续优化Agent决策能力

## 使用方法

### 前端访问
1. 进入"复盘报告"Tab
2. 选择时间周期（每日/每周/每月等）
3. 查看预测准确率和差异分析
4. 阅读Agent反馈和改进建议
5. 点击"生成复盘报告文档"保存

### 后端生成
```bash
cd backend
python review_report_generator.py
```

# 宏观环境分析功能

## 功能概述

宏观环境分析模块每日自动生成经济/政治/战争对股票市场影响的分析文章，采用金字塔原理结构，由浅入深呈现。

## 金字塔原理结构

### 第一层：核心结论
- 一句话总结文章核心观点
- 直接给出投资建议

### 第二层：关键论点
- 3个支撑论点
- 每个论点详细阐述
- 逻辑清晰，层次分明

### 第三层：支撑数据
- 具体数据和事实
- 可验证的数据来源
- 增强说服力

## 文章类别

### 1. 经济类
- 美联储货币政策
- 全球通胀趋势
- 汇率波动
- 经济增长前景

### 2. 政治类
- 中美关系
- 地缘政治
- 政策变化
- 国际关系

### 3. 战争类
- 俄乌冲突
- 中东局势
- 能源市场影响
- 避险情绪

## 文章结构

每篇文章包含：

1. **标题和摘要**
2. **核心结论**（金字塔顶层）
3. **关键论点**（3个）
4. **支撑数据**（5条）
5. **影响分析**
   - 正面影响板块
   - 负面影响板块
   - 中性影响板块
6. **投资建议**（4-5条）
7. **参考资料**（带链接）

## 定时生成

建议配置每日定时任务：

```bash
# crontab -e
0 8 * * * cd /path/to/backend && python macro_article_generator.py
```

- 每日早上8点自动生成
- 轮换经济/政治/战争主题
- 保留最近30篇文章
